                    DIfferences between C and C++                     
                
                
                    

Though they both have that letter C, C and C++ are not similiar, they have the following diffrences                         

                                
  1. C was developed by Dennis Ritchiein 1972 while C++ was developed by Bjarne Stroustrup in around 1979
    2.                             
  2. C is a procedural programming language while C++ supports both object oriented and procedural programming paradigms
    3.                             
  3. As C does not support the OOPs concept so it has no support for polymorphism, encapsulation, and inheritance while C++ has support for polymorphism, encapsulation, and inheritance as it is being an object-oriented programming language
    4.                             
  4. As C does not support encapsulation so data behave as a free entity and can be manipulated by outside code while  in the case of C++ encapsulation hides the data to ensure that data structures and operators are used as intended.
    5.                             
  5. C in general known as function-driven language while C++ is known as object driven language.
    6.                         
                    

                
                         
                             
                    Structure of the C language                     
                
                
                    

Below is a skeleton structure of the C langauge and every C program must follow this flow:

                                                    preprocessor directive
Globall variable declaration(optional)
main function(){
//codes/syntax
return type
}                                              

                    

preprocessor directive

                    

It tells the compiler of the type of header file(s) to be used in the program, usually started by #include and the inside                          opening and closing angle brackets, we write the header file with a dot h, for example, to include the input                          output header file, we write #include <stdio.h>this will tell the compiler tom include standardinput output header files.                     

                    

Global variable declaration

                    

In almost all programming languages, you will find two types of variables, LOCAL VARIABLES and GLOBAL VARIABLES, local variables are variables                          that are created inside and used inside a function (we will talk more about functions later if you have no idea of what they are) while global variables are variables                         that are created outside functions and can be used anywhere in the program in any function, in C programming, all global variables should come immeadiately after the preprocessor directive just as shown in the skeleton structure.                     

                    

The main function

                    

It is an inbuilt function in C programming where "the actual execution of program takes place", it is enclosed with a pair of curly brackets and it can be anywhere in the program,                         however, all your C programs should have only one main function.                     

                    

inside the main function is now where all our C codes come in and where functions are called.
Lastly, all C statements must be terminated with a semi colon except the preprocessor directive and the main function

                
                         
                             
                    Hello world in C                     
                
                
                    #include <stdio.h>
int main(){
printf("Hello wolrd");
return 0;
}                     

remember, all C statements inside the main function or any other function must be terminated with a semi colon

                    

printf() function

                    

This function is used to printtext or information to the console, the syntax is to write what you want printed on the console inside the circle brackets enclosed with double quotes.

                    comments in C programming                     
                
                
                    

A comment is a line of code that cannot be read or executed by the compiler, in other words, the compiler becomes blind upon seeing any comment and is able to see again when it comes out of the comment, it is as if that line of code never exists, comments are a great way to debug our code in that, when your code has an error and the compiler is telling you that a certain line has an error, instead of rushing to delete that line of code, just comment it out since you have maybe forgotten what the line of code was supposed to do, we can also use comments to tell other programmers what our code does so that when they reading through our code, instead of figuring out on themselves what the code does, they just see that throught the comments we created for them. There are two types of comments in C, single line comments denoted by // and multiline comments denoted by /* and */

                    Variables in C programming                     
                
                
                    

A Variable is a data name which is used to store some data value or storing program computations and results.

                    

In any programming language, before using a variable, you must DECLARE and INITIALISE it, declaration is the most important as initialisation can be done anywhere anytime.

                    

Variable declaration in C programming

                    

To declare a variable in C programming, we start with the data type of the variable, followed by the name of the variable terminated by a semicolon.

                    

The data type should reflect what the variable is supposed to hold, for example if you want to store a digit, you use int, if you want a character, you use char, etc.

                    
example
int number;
in this example, we have created a variable called number which is used for holding non-decimal values, since we have given it the data type of int, we have then terminated with a semi-colon, always remember this rule.                     

Rules for giving variable names in C

                    

In our example above, number is our variable name; there are several rules that govern the "making" of these names in C programming as follows:                         

                                
  1. Variable names should not start with numbers or be purely made up of numbers, for example;                                 
                                          
    • int 123;-------is wrong
      •                                     
    • int 1number;------is wrong.
      •                                     
    • int number1;--------is right
      •                                 
                                
    2.                             
  2. Variable names should not have any special characters apart from underscore, for example;                                 
                                          
    • char student-grade;------is wrong
      •                                     
    • char student.grade;--------is wrong
      •                                     
    • char student_grade; --------is correct
      •                                 
                                
    3.                             
  3. Variable name should not contain spaces, for example;                                 
                                          
    • char student grade;-------is wrong
      •                                     
    • char studentgrade;-------is correct
      •                                 
                                
    4.                             
  4. Variable names should not be keywords, for example;                                 
                                          
    • int double;-------is wrong
      •                                     
    • int float;--------is also wrong
      •                                 
                                
    5.                         
                    

                    

Variable initialisation in C programming

                    

Initialising a variable refers to assigning an initial value to a variable you have created/declared.

                    

The syntax is to declare the variable in a normal way but instead of terminating it with a semi colon, you write the assignment operator (=) and write th value of the variable to the right side and then terminate with a semi-colon, for example                         
int number=10;
char grade='A';
                        You can also declare a variable in the normal way then initialise it later as shown:
int num;
num=10;
                        

C does not support string data type, therefore, to declare a string variable, we use char following the procedure below:
                        

                                
  1. write the char keyword.
    2.                             
  2. write your variable name followed by square brackets, inside the square brackets, write hoe many character the dtring should hold.
    3.                             
  3. If you are just declaring, terminate with a semicolon but to initialise, write the assignment operator, then write the value inside double quotes, for example.                                 
                                          
    • char name[6]="dennis";
      •                                     
    • char studentName[30];
      •                                     
    • char programmer[9]="zaphenath";
      •                                 
                                
    4.                         
                        

                    

                
                         
                             
                    Keywords in C programming                     
                
                

keywords are words that are reserved for doing specific purposes, these words are predefined and are always written in lowercase , they cannot be used for anything else.                     keywords in C are int, short, signed, unsigned, default, volatile, float, long, double, break, continue, typedef, static, do, for, union, return, while, extern, register, enum, case, goto, struct, char, auto, etc.
